# Chat REST API

Spring Boot REST API application for managing users and chat messages, with JWT authentication and role-based access control.

---

## Tech Stack

- Java 17
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA
- H2 (Test DB)
- Gradle
- JUnit 5

---
